## 기술 스택

- UI: Compose for Desktop
- 로컬 DB: SQLite or Mysql (변경 가능)
- 서버: Spring Boot

## 서비스 소개

1. 뽀모도로 타이머

- 시작 / 일시정지 / 종료
- 세션 자동 기록

2. 집중 데이터

- 공부 시간
- 집중 점수 (자가 입력)

3. 대시보드

- 시간대별 집중도
- 요일별 패턴
- 연속 집중 시간

4. 인사이트 “너는 오후 9시에 집중 잘되는 편”
